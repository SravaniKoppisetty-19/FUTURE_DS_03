# Marketing Funnel & Conversion Performance Analysis Dashboard

## Project Overview
This project analyzes marketing funnel performance and conversion rates across multiple marketing channels using Microsoft Excel.

## Key Metrics
- Total Visitors: 28,260
- Total Leads: 3,782
- Total Customers: 355
- Overall Conversion Rate: 1.26%

## Dashboard Features
- KPI Cards
- Marketing Funnel Visualization
- Conversion Rate Analysis
- Customers by Marketing Channel
- Key Business Insights

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Visualization
